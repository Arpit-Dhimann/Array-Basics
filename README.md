# Array-Basics

A module focused on mastering Arrays in Java. Covering 
declaration, user input, and basic searching algorithms.

### 🚀 Programs Implemented
- ✅ **Array_Declaration.java**: Two approaches to declare and initialize arrays.
- ✅ **Array_Input_Output.java**: Dynamic array input/output using Scanner.
- ✅ **Linear_Search.java**: Algorithm to search for a number and return its index.
- ✅ **Practice_Questions_1.java**: Array of names input and display logic.
- ✅ **Practice_Questions_2.java**: Finding min/max values in an integer array.
- 📝 **NOTES.md**: Key concepts and time complexity analysis.

## 🛠️ Concepts Covered
- Indexing (0-based)
- Array Traversal using `for` loops
- Linear Search Algorithm — O(n)
