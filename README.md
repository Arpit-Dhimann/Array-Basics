# Array-Basics

A module focused on mastering Arrays in Java. Covering 
declaration, user input, and basic searching algorithms.

## 🚀 Programs Implemented

- ✅ **Array_Declaration.java**: Two approaches to declare 
and initialize arrays in Java.
- ✅ **Array_Input_Output.java**: Dynamic array input and 
output handling using Scanner.
- ✅ **Linear_Search.java**: Algorithm to search for a 
number and return its index.
- 📝 **NOTES.md**: Key concepts and time complexity of 
array operations.

## 🛠️ Concepts Covered
- Indexing (0-based)
- Array Traversal using `for` loops
- Linear Search Algorithm — O(n)
